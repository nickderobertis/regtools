regtools.fe.output module¶

regtools.fe.output.add_fixed_effects_rows(summ_df, fixed_effect_dict)[source]¶

summ_df: regression summary df where models are columns fixed_effect_dict: dictionary where keys are names of fixed effects and values are a single boolean for whether to include the fixed effect for all models, or a list of booleans for whether to include the fixed effect for each model, e.g. for a 3 model summ_df:

fixed_effect_dict = {

‘Industry’: True, ‘Time’: [False, False, True]

}
